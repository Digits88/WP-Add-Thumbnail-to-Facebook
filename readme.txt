== Description ==

Many themes have the problem that they do not generate an appropriate thumbnail with according meta tag in your page head for use with Facebook's "Share" function.This plugin will add ogg meta tags and perfect thumbnail to your share on Facebook.

== Installation ==

1. Check if "add_theme_support('post-thumbnails');" is existing in your theme's functions.php file
2. If not, add this line!
3. Download the plugin, unzip it and upload the "wp add thumbnail to facebook" folder to the "/wp-content/plugins/" directory
4. Activate the plugin and regenerate your thumbnails (Using the "AJAX Thumbnail Rebuild" plugin)
5. Enjoy!
